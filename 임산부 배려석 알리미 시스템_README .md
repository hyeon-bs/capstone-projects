# 🤰 임산부 배려석 알리미 시스템 (Pregnant Seat Assistance System)

## 📝 프로젝트 개요
대중교통 내 임산부 배려석 이용 문제를 개선하기 위해, Bluetooth 기반 근거리 통신 기술과 Raspberry Pi 환경을 활용하여 임산부 사용자의 접근 여부를 감지하고 좌석 알림 서비스를 제공하는 시스템을 설계했습니다.

기존 지하철 “핑크라이트” 시스템의 비콘(BLE Beacon) 기반 구조를 분석하고, 별도 전용 비콘 없이 스마트폰과 Raspberry Pi를 활용하는 저비용 대체 구조를 제안했습니다.

## 📅 프로젝트 기간
* 2018

## 💻 기술 스택
Platform: Raspberry Pi
Communication: Bluetooth BLE
OS: Linux
Development Environment: Raspberry Pi / Linux Terminal

## 🎯 프로젝트 목적 및 내용

### 1. BLE 기반 사용자 감지 시스템 설계
Bluetooth 신호를 활용하여 임산부 사용자의 접근 여부를 감지하는 구조를 설계했습니다.
* Raspberry Pi를 BLE 송수신 장치로 활용
* Linux 기반 BlueZ 라이브러리 환경 구성
* Bluetooth 페어링 및 신호 송수신 흐름 구현


### 2. 기존 비콘 시스템의 한계 분석 및 개선 방향 제안
당시 운영되던 “핑크라이트” 시스템의 구조적 한계를 분석하고 개선 아이디어를 설계했습니다.
* 별도 비콘 분실 및 반납 문제
* 유지보수 비용 증가
* 초기 임산부 인식 한계

이를 개선하기 위해:
*스마트폰 기반 접근 구조
*Raspberry Pi 기반 저비용 센서 구조를 제안했습니다.

### 3. 실제 기술 흐름과의 연관성 분석
프로젝트 수행 이후, 국내 대중교통 환경에서는 BLE·비콘·IoT 기반 교통약자 지원 기술이 실제로 확대 적용되기 시작했습니다.

당시 프로젝트에서 제안했던:

* BLE 기반 사용자 감지
* 스마트 디바이스 연동
* 교통약자 접근성 향상

구조가 이후 스마트 교통 인프라 및 IoT 기반 공공서비스 흐름과 유사한 방향으로 발전하고 있다는 점에서 의미 있는 선행 학습 경험이 되었습니다.

## ⚠️ 프로젝트 문제점 및 한계
1. 실제 대중교통 환경에서의 대규모 BLE 간섭 테스트는 수행하지 못했습니다.
2. Raspberry Pi와 스마트폰 간 거리 및 장애물 환경에 따라 신호 안정성이 달라질 가능성이 존재했습니다.
3. 실제 서비스 수준의 사용자 인증 및 보안 구조까지는 확장되지 못했습니다.
